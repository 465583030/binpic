# binpic

Create a picture from a file.

```
$ binpic /bin/ls
```

Encode a binary as [grayscale image](https://golang.org/pkg/image/#Gray), optionally resize.

```
$ binpic -h
Usage of binpic:
  -o string
        output file, will be a PNG (default "output.png")
  -resize string
        resize, if set (default "0x0")
```
